## 故事

你是一名學生，剛放學準備回家。
最近有不法分子販賣太空油，
這種毒品一旦接觸，
會對身體造成嚴重影響。
你能成功避開太空油，
安全順利地回到家嗎？

---

## 玩法

按下 **`鍵盤`/`畫面`** 跳躍，
利用跳躍避開太空油，
安全順利地回家。
